# Data folder

This folder intentionally does not contain the raw violation dataset, since it was provided under the Gridlock Hackathon's data-usage terms.

To run the notebook:
1. Obtain the raw CSV from the hackathon organizers (file expected by the notebook: `jan to may police violation_anonymized791b166.csv`).
2. Place it in this `data/` folder, or directly next to the notebook.
3. The notebook auto-detects the file in either location — no path changes needed.
